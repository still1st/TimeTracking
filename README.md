# TimeTracking
Приложение для создания ежемесячных табелей для медицинского персонала.
Две группы медперсонала:
1. Младший медперсонал
2. Средний медперсонал и врачи

Для каждой группы есть норма часов в день.

Исходя из нормы часов в день, праздников и выходных строится табель за указанный месяц.
Табель можно сохранять в виде excel-документа.
